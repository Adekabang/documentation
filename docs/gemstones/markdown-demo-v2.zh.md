## 演示模式

> 这是一个例子。 格式精良。

在大多数情况下，它是这样直截了当的。

您有时需要显示 `命令`

或多个命令：

在其他时候，您需要子弹或编号列表：

```
dnf 安装my_stapler
dnf 更新 my_pen铅笔。
```

你可能需要警告：

- 我相信你有我的订书机
- 如果没有，请至少返回我的铅笔。

1. 这里是你的铅笔破损了
2. 尖锐化是无用的

你可能需要警告：

!!! 提示

    铅笔和主食确实是老学校。

当命令有多个选项时，其次数多于不多。 或者你需要列出具体内容，你可能想要使用一个表来识别事物：

| 工具       | 使用          | 发布工程说明              |
| -------- | ----------- | ------------------- |
| 铅笔数      | 撰写或打印       | 常用笔替换               |
| 笔数       | 撰写或打印       | 常用样式表替换             |
| stylus   | 在电子设备上写作或打印 | 有时被键盘替换             |
| keyboard | 在电子设备上写作或打印 | 永远不会替换 - 直到满载食物时才使用 |
